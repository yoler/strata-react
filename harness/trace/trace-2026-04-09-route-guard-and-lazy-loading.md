# 2026-04-09 路由守卫与懒加载记录

## 背景

在架构审查后，决定继续补齐两项模板层能力：

1. 将登录页中的已登录重定向逻辑提升到路由层
2. 给页面入口增加最基础的路由懒加载能力

## 本次调整

1. 新增 `GuestRoute`
2. 登录页去掉内联 `Navigate`
3. 页面入口增加默认导出，供 `React.lazy` 使用
4. router 中通过统一 `RouteLoading` 包裹懒加载页面

## 说明

- 这次优先补的是“最小可执行版本”的路由懒加载
- 目前仍然保留显式 route 配置，不引入更重的 route-module 组织方式
- 后续如果页面规模继续增长，可以再往更细粒度的路由分片推进
